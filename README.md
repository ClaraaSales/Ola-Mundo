# Projeto
A Etec Bartolomeu Bueno da Silva realizou, no ano passado, a feira anual de tecnologia, conhecida por Bartotec. Nós, alunos do segundo ano do curso de Informática tivemos como proposta o desenvolvimento de um protótipo de site voltado à abordagem de problemas relevantes do mundo atual.
O projeto envolveu o uso de linguagens de programação e a implementação de um banco de dados.
